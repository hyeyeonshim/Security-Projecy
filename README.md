# Security-Project
- 프라이버시 보존 데이터 학습을 위한 고효율 동형 암호 기법 논문에 대한 코드입니다.
  1) homomorphic_machinelearning_conventional.ipynb
    - 데이터셋 전체를 동형암호로 암호화하여 머신러닝을 수행한 경우 
  2) homomorphic_machinelearning_proposed.ipynb
    - 데이터셋의 일부를 동형암호로 암호화하여 머신러닝을 수행한 경우

[참고]
- Github에 업로드된 버전의 코드가 논문의 표를 만들 때 사용한 설정(parameter)을 사용하고 있지 않을 수 있습니다. (후속 연구 준비로 인하여 코드 파라미터를 수정함)
- 동형암호화 노이즈로 인하여 매 실행마다 다른 결과가 발생합니다.
